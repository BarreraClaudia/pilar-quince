## Sections

- **Ribbon** — top banner with event tagline and date
- **Hero** — full-screen background image with animated floating petals and Pilar's name
- **Message** — love note from parents with decorative quotation marks
- **Photo Gallery** — mosaic grid of 13 photos with hover effects and lightbox
- **Event Details** — date, church, and reception info with Google Maps buttons
- **Countdown** — live countdown timer to August 1, 2026 at 3:00 PM
- **Footer** — monogram and closing line

## Fonts

Both loaded from Google Fonts:

- **Cormorant Garamond** — headings, quotes, names (elegant serif)
- **Jost** — labels, body text, UI elements (clean sans-serif)

---

## Features

- Animated floating petals on the hero section
- Parallax scrolling effect on the hero background
- Scroll-triggered fade-in animations throughout
- Photo gallery with hover zoom and pink overlay
- Click-to-open lightbox for full-size photos
- Live countdown timer with flip animation
- Google Maps buttons on venue cards
- Responsive design for mobile, tablet, and desktop

---

## Browser Support

Works in all modern browsers — Chrome, Safari, Firefox, and Edge. The site uses standard CSS and vanilla JavaScript with no dependencies.
